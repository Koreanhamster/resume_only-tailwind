## 기술스택
- HTML + Tailwind (CSS없이 오직 Tailwind만 사용)

## 배운점 & 아쉬운점
- Tailwind의 문법에 어느정도 익숙해졌다. 생각보다 간단하고 할 수 있는게 다양하다. 
- 확실히 유지보수는 힘들다. 같은 속성을 가지고있는 모든 요소들의 클래스를 변경해줘야한다.
- 미디어쿼리를 적용할 수 없어 아쉬웠다. CSS와 적절히 섞어 사용하면 정말 빠르고 편하게 웹을 만들 수 있겠다!
